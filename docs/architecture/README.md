# OVERSCITE Global — Architecture Documentation

## Position Statement

OVERSCITE Global is a **governed application workspace**. It is a connected consumer and binder of SCINGULAR Prime Intelligence — not the source of it.

> **SCINGULAR remains outside and above the app. Prime Intelligence is not app-specific. ScingOS remains the Prime Orchestrator. OVERSCITE Global is a connected governed application surface.**

### System Hierarchy

```
SCINGULAR Ecosystem / ScingOS Prime
    ├── ScingIntelligence
    ├── LARIIntelligence
    └── BANEIntelligence
            ↓  (governed SICP / NeuralBind connection)
     OVERSCITE Global
       ├── App bindings
       ├── App overlays
       └── Local workflows
```
