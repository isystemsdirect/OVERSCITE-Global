# LARI-IPN Anomaly Classification

## The Engine
Anomaly detection does not act as punishment and holds zero execution capability automatically.

### Anomaly Taxonomy
- **Identity Anomaly**: Signature/Key drift detection (`CRITICAL`).
- **Conflict Pressure Anomaly**: Structural egress routing issues like VPN conflicts (`HIGH`).
- **Heartbeat Anomaly**: Dead-node polling behavior (`MEDIUM`).
- **Burst Anomaly**: High packet velocity variance (`MEDIUM`).

LARI ensures all anomalies trace directly back to structural parameters, avoiding opaque deep-learned assertions without associated Evidence References (`evidenceSummary`).
