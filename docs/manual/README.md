## About This Manual

This document is the definitive operator reference for the OVERSCITE Global platform. It covers system philosophy, architecture, interface operations, environmental intelligence, data governance, operational workflows, scheduling, audit enforcement, and user responsibility.

Every statement in this manual reflects the actual implemented behavior of the OVERSCITE system. Where features are under active development or operating with simulated data, truth-state markers indicate the precise maturity level.

**Audience**: All OVERSCITE operators — from first-time users to advanced field professionals.

## Truth-State Legend

Throughout this manual, truth-state badges indicate the maturity of each described feature:

- **[LIVE]** — Confirmed, operational, and connected to real-world execution.
- **[PARTIAL]** — Scaffolded or incomplete systems under active development.
- **[MOCK]** — Synthetic data provided for structural simulation and UI validation.
- **[CANDIDATE]** — Unverified logic or proposals requiring evaluation.
- **[ACCEPTED]** — Evaluated and approved proposals locked into canon.
