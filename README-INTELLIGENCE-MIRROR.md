# ScingIntelligence — Intelligence Mirror Branch

## What This Branch Is

**ScingIntelligence** is a governed intelligence mirror branch inside the `OVERSCITE-Global` repository. It provides a repo-side access surface for Scing-linked operations, enabling agent locality and structured communication with the SCINGULAR Prime Intelligence ecosystem.

This branch mirrors interface/orchestration intelligence access. It does not contain, replace, or replicate the Prime Intelligence source.

## What This Branch Is Not

- **Not** the origin of Scing Prime Intelligence
- **Not** the Prime Orchestrator (ScingOS remains Ground Zero)
- **Not** a co-prime or authority peer to ScingOS
- **Not** an OVERSCITE application runtime surface
- **Not** a substitute for the SCINGULAR ecosystem

This branch provides a governed mirror surface where:

1. **InterCom/** — Structured communication channels (inbound, outbound, shared, protocol) for intelligence transport between OVERSCITE-Global agents and the external Prime Intelligence stack
